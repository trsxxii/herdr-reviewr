# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version: be kind, assume good faith, critique code and ideas rather than people, and
keep the space welcoming for reviewers and contributors at every experience level.

Report unacceptable behavior privately to the maintainer through
[GitHub](https://github.com/persiyanov). Reports are handled confidentially.
